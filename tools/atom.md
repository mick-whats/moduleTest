## Packages

### [git\-plus](https://atom.io/packages/git-plus)
パレット表示
- `Cmd-Shift-H` on MacOS
- `Ctrl-Shift-H` on Windows + Linux

基本的にはパレットを開いてコマンドを入力するだけ。  
コマンドはインクリメンタルサーチが効く。  
例えば`acp`と入力すると候補に`Add And Commit And Push`と表示される。
